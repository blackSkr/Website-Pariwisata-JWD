==============================
apabila user telah upload Laporan / berkas otomati akan mendapatkan status pending yg akan di crosscheck oleh admin
user : 
- registrasi menggunakan nim, nama, prodi, alamat, email, wa, Password
- login menggunakan nim dan password
- wajib inputkan/upload gambar/scan foto berkas secara jelas dan lengkap

admin : 
- registrasi manual dengan halaman regis sendiri yang berisi noadmin, username, email, password
- dapat melihat semua laporan(berkas) yang masuk tanpa terkecuali
- dapat edit status laporan(berkas) menjadi status yang diinginkan
 contoh : pending -> belum di tindak, progress -> lagi di crosscheck, verified -> diterima
